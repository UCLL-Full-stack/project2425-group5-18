# Car Vault


Het project betreft een site waarop gebruikers kunnen inloggen en hun collectie van auto's kunnen digitaliseren. Gebruikers kunnen auto's aan hun collectie toevoegen door het invullen van een formulier en dit later ook aanpassen. Andere gebruikers kunnen auto's op hun "verlanglijst" zetten en contact op nemen met auto's die "te koop" staan. Gebruikers voegen een adres en contactgegevens in die daarna gedeeld kunnen worden wanneer iemand interesse heeft in de auto. Later worden prestaties toegevoegd aan de gebruikers en statistieken (grootste collectie, meeste boetes, ...)
